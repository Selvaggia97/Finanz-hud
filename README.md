Mein privates Finanz-HUD v3 (Desktop-integrierter Button)
===================================================

1. ZIP-DATEI ENTPACKEN:
   Entpacke diesen Ordner komplett an einen Ort deiner Wahl (z.B. in deine Dokumente).

2. STARTEN:
   Mache einen Doppelklick auf "starten.bat". 
   Du siehst zuerst kein großes Fenster. Schaue stattdessen an den RECHTEN Bildschirmrand:
   Dort klebt jetzt ein kleiner, runder, blauer Button mit einem "€"-Symbol.

3. SMARTES VERHALTEN (WICHTIG):
   - Der "€"-Button verhält sich wie ein Desktop-Icon! Er klebt am Hintergrund und wird von
     geöffneten Browsern oder Programmen verdeckt, damit er dich beim Arbeiten nicht stört.
   - Wenn du darauf klickst, klappt sich das komplette Finanz-HUD auf und setzt sich in den Vordergrund.
   - Klickst du im HUD auf ">", klappt es wieder zu und der Button verschwindet wieder dezent in den Hintergrund.

4. BEDIENUNG & FIXKOSTEN:
   - Trage Beträge schnell im HUD ein (z.B. "12,50", Typ wählen, Kategorie wählen, Buchen).
   - Klicke oben rechts im HUD auf das Zahnrad-Symbol (⚙). Dort kannst du deine Fixkosten eintragen, 
     Transparenz und Höhe anpassen oder alte Buchungen löschen.

5. AUTOSTART:
   Klicke im Zahnrad-Menü einfach auf "Autostart ein/aus". Das HUD startet ab jetzt bei jedem Windows-Start vollautomatisch im Hintergrund als Desktop-Gadget!

6. BEENDEN:
   Öffne das Zahnrad-Menü (⚙) und klicke ganz unten auf den roten Button "Komplettes Programm beenden".
